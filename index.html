API Gateway — Cloudflare Edge
Versioned REST API endpoints with JWT, caching, and rate limiting
Authentication Flow
POST /api/v1/auth/login
  → Validate credentials
  → Generate JWT (HS256)
  → Return token + user

Authorization: Bearer <token>
  → Verify signature
  → Check expiry (24h)
  → Extract user claims
  → Allow / Deny (401)
